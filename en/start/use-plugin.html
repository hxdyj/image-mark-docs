How to Use Plugins ​

ImageMark will automatically use the built-in shape , selection , history , shortcut plugins when importing. If you do not wish to use them automatically, you can use the unuseDefaultPlugin method to disable.

Example ​

Class Method ​

ts
import ImageMark, { SelectionPlugin } from 'mark-img'

const imgMark = new ImageMark({
	el: '#container',
	src: './example.jpg',
})

imgMark.addPlugin(SelectionPlugin, {
	// plugin options
})

Function Method ​

ts
import ImageMark, { SelectionPlugin } from 'mark-img'

const imgMark = new ImageMark({
	el: '#container',
	src: './example.jpg',
})

imgMark.addPlugin(imageMarkInstance => {
	return new SelectionPlugin(imageMarkInstance, {
		// plugin options
	})
})
